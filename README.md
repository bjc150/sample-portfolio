# Bing Janchoo - Portfolio Website

A professional, responsive portfolio website for Bing Janchoo, Fixed Income Fund Manager. This project showcases expertise in credit risk management, yield curve analysis, and macro-economic data visualization.

## 🚀 Features

- **Dark Theme:** A modern, professional dark-mode aesthetic for high readability.
- **Scroll Reveal Animations:** Smooth entrance animations using the Intersection Observer API for an interactive user experience.
- **Responsive Design:** Optimized for desktop, tablet, and mobile devices.
- **SEO Optimized:** Includes meta tags for search engines and social media sharing (Open Graph).
- **Fast Loading:** Built with vanilla HTML, CSS, and JavaScript for maximum performance.

## 🛠️ Tech Stack

- **HTML5:** Semantic structure.
- **CSS3:** Custom properties (variables), Flexbox, Grid, and Animations.
- **JavaScript (ES6+):** Intersection Observer for scroll animations and smooth navigation.
- **Typography:** [Roboto Mono](https://fonts.google.com/specimen/Roboto+Mono) via Google Fonts.

## 📁 Project Structure

- `index.html`: The main landing page.
- `styles.css`: All styling and theme configurations.
- `script.js`: Interactive elements and animations.
- `project_plan.md`: The development roadmap.
- `project_summary.md`: A summary of the project's history and current status.

## 💻 Local Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/bjc150/sample-portfolio.git
   ```
2. **Navigate to the directory:**
   ```bash
   cd sample-portfolio
   ```
3. **Open `index.html`:**
   Simply open the `index.html` file in any modern web browser.

## 🌐 Deployment

The site is hosted on GitHub Pages. To deploy your own version:
1. Push the code to a GitHub repository.
2. Go to **Settings > Pages**.
3. Select the `main` branch and `/ (root)` folder.
4. Click **Save**.
