# Portfolio Website Implementation Plan

## Objective
Build and deploy a personal portfolio website for Bing Janchoo, Fixed Income Fund Manager, using simple HTML, CSS, and JavaScript. The site will be hosted on GitHub Pages.

## Key Files & Context
*   `index.html`: The main structure of the single-page portfolio.
*   `styles.css`: Stylesheet implementing the dark theme and typography.
*   `script.js`: (Optional) Lightweight JavaScript for interactivity (e.g., smooth scrolling).
*   `.gitignore`: Basic git ignores.

## Specifications
*   **Theme:** Dark theme (e.g., background `#121212`, text `#E0E0E0`).
*   **Typography:** 'Roboto Mono' font from Google Fonts.
*   **Hero Section:** Prominently displaying "Bing Janchoo" and "Fixed Income Fund Manager".
*   **Projects Section:** Three relevant project placeholders (e.g., "Yield Curve Analysis Tool", "Macro-economic Data Dashboard", "Credit Risk Modeling").
*   **Contact Section:** Email, LinkedIn, and GitHub links.

## Implementation Steps
1.  **Project Initialization:**
    *   Create `index.html`, `styles.css`, and `script.js` in the root directory.
    *   Initialize a local Git repository.
2.  **HTML Structure (`index.html`):**
    *   Set up semantic HTML5 boilerplate.
    *   Import 'Roboto Mono' from Google Fonts.
    *   Create `<header>`, `<section id="hero">`, `<section id="projects">`, `<section id="contact">`, and `<footer>`.
3.  **Styling (`styles.css`):**
    *   Apply the dark theme color palette.
    *   Set the global font family to 'Roboto Mono', monospace.
    *   Use CSS Flexbox/Grid for a clean, responsive layout across desktop and mobile devices.
4.  **Content Population:**
    *   Fill in the Hero section with the name and title.
    *   Add 3 card-style containers for the projects.
    *   Add contact information and links.
5.  **GitHub Pages Deployment:**
    *   Provide commands to commit the code.
    *   Provide instructions on pushing to a GitHub repository and enabling GitHub Pages from the repository settings.

## Verification & Testing
*   Open `index.html` in a local web browser.
*   Verify the dark theme and 'Roboto Mono' font are applied correctly.
*   Ensure the layout is responsive and looks good on different screen sizes.
*   Test all navigation and external contact links.
